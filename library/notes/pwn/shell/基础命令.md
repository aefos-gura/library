## 文件和目录操作

- ls - 列出目录内容
    
    - ls -l：详细列表
        
    - ls -a：显示隐藏文件
        
- cd [目录] - 切换目录
    
    - cd ..：返回上级目录
        
    - cd ~：返回家目录
        
- pwd - 显示当前工作目录路径
    
- mkdir [目录名] - 创建目录
    
- rm [文件名] - 删除文件
    
    - rm -r [目录名]：删除目录
        
    - rm -f：强制删除
        
- cp [源文件] [目标] - 复制文件/目录
    
    - cp -r：递归复制目录
        
- mv [源文件] [目标] - 移动/重命名文件/目录
    
- touch [文件名] - 创建空文件或更新时间戳
    
- cat [文件名] - 查看文件内容
    
    - cat > [文件名]：创建并写入文件
        
- less [文件名] - 分页查看文件内容
    
- head [文件名] - 显示文件开头
    
    - head -n 5：指定行数
        
- tail [文件名] - 显示文件末尾
    
    - tail -f：实时监控
        

## 文件权限和所有权

- chmod [权限] [文件名] - 修改文件权限
    
    - 例：chmod 755 文件 或 chmod u+x 文件
        
- chown [用户] [文件名] - 更改文件所有者
    
    - chown -R：递归更改目录
        
- ls -l - 查看权限和所有者
    

## 查找和搜索

- find [路径] -name [文件名] - 查找文件/目录
    
- grep [模式] [文件名] - 搜索文件内容
    
    - grep -r：递归搜索
        
- locate [文件名] - 快速查找文件
    
    - 需更新数据库：updatedb
        

## 磁盘和系统信息

- df -h - 显示磁盘使用情况
    
- du -sh [目录] - 显示目录/文件大小
    
- free -m - 查看内存使用
    
- top - 实时查看进程和资源
    
    - 替代：htop（需安装）
        
- uname -a - 显示系统信息
    

## 进程管理

- ps aux - 查看当前进程
    
- kill [PID] - 终止进程
    
    - kill -9：强制终止
        
- jobs - 查看后台任务
    
- bg - 后台运行暂停任务
    
- fg - 切换后台任务到前台
    

## 文本处理

- echo [文本] - 输出文本
    
    - echo $变量：显示变量值
        
- cut - 提取行中特定部分
    
- sort - 排序文本行
    
- uniq - 去除重复行（常与 sort 配合）
    
- wc - 统计行数、字数、字符数
    
    - wc -l：仅统计行数
        

## 管道和重定向

- | - 管道，连接命令输出和输入
    
- > - 重定向输出到文件（覆盖）
    
- >> - 重定向输出到文件（追加）
    
- < - 从文件读取输入
    
- tee - 输出到文件和终端

## 系统管理

- man [命令] - 查看命令帮助
    
- history - 显示命令历史

## 环境变量

- env - 列出环境变量
    
- export [变量名]=[值] - 设置环境变量

- tar -xvf [归档名].tar - 解压 tar 归档

## 通配符的区别

- **`*` (星号)**：匹配**零个或多个**任意字符。
    
    - `a*.txt` 匹配 `a.txt`, `ab.txt`, `abc.txt`。
        
- **`?` (问号)**：匹配**恰好一个**任意字符。
    
    - `data?.log` 匹配 `data1.log`, `dataA.log`，但不匹配 `data.log` 或 `data10.log`。
        
- **`[]` (方括号)**：匹配方括号内指定的**任意一个**字符。
    
    - `[abc].txt` 匹配 `a.txt`, `b.txt`, `c.txt`。
        
    - `[0-9].log` 匹配 `1.log`, `2.log` 等，但不匹配 `a.log`。
